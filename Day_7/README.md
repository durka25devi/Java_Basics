# Tic Tac Toe Game (Java)
## Overview

This is a simple Tic Tac Toe (XO) game built using Java. Two players can play the game by entering row and column numbers. The program checks for a winner or draw after each move.

## Features

Two-player game (X and O)
3Ã—3 game board using 2D array
Checks winner (row, column, diagonal)
Detects draw condition
Console-based interface

## How to Run

### Compile:

javac tic_tac_toe_game.java

### Run:

java tic_tac_toe_game

## How to Play

Players enter row and column values from 0 to 2
Player X starts first
Player who gets 3 in a row, column, or diagonal wins
If all cells are filled, the game is a draw
