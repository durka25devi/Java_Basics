# STRING METHODS

1. length() – returns number of characters
2. charAt(int index) – returns character at index
3. toUpperCase() – converts to uppercase
4. toLowerCase() – converts to lowercase
5. trim() – removes starting and ending spaces
6. equals(String s) – compares content of two strings
7. equalsIgnoreCase(String s) – compares ignoring case
8. compareTo(String s) – lexicographical comparison
9. compareToIgnoreCase(String s) – comparison ignoring case
10. concat(String s) – joins two strings
11. contains(CharSequence s) – checks substring
12. startsWith(String prefix) – checks starting text
13. endsWith(String suffix) – checks ending text
14. indexOf(char/String) – first occurrence index
15. lastIndexOf(char/String) – last occurrence index
16. replace(char, char) – replaces characters
17. replace(CharSequence, CharSequence) – replaces text
18. replaceAll(String regex, String) – regex replacement
19. replaceFirst(String regex, String) – first regex replacement
20. substring(int begin) – substring from index
21. substring(int begin, int end) – substring between indexes
22. split(String regex) – splits into array
23. isEmpty() – checks if empty
24. isBlank() – checks if empty or spaces
25. toCharArray() – converts to char array
26. getBytes() – converts to byte array
27. valueOf(any) – converts data to string
28. Join(CharSequence, String...) – joins multiple strings
29. format(String, args) – formatted string
30. matches(String regex) – regex match check
31. codePointAt(int index) – Unicode of charater
32. codePointCount(int, int) – Unicode count
33. regionMatches(...) – compares part of strings
34. subSequence(int, int) – returns character sequence
